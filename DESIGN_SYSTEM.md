# Design System - 5Brothers Legacy Initiative

## Color Palette (Based on Logo Colors)

### Primary Colors
- **Navy**: `#0B334A` - Primary brand color from logo, trust, professionalism, stability
- **Navy Dark**: `#082530` - Depth, authority, hover states
- **Navy Light**: `#0F4A6A` - Lighter shade for gradients and hover states

### Accent Colors
- **Red**: `#DE3C3A` - Accent color from logo, urgency, action, call-to-action buttons
- **Red Dark**: `#C22A2A` - Darker shade for depth
- **Red Light**: `#E85A5A` - Lighter shade for hover states
- **Success Green**: `#10B981` (Emerald-500) - Growth, positive impact
- **Warm Orange**: `#F59E0B` (Amber-500) - Warmth, community
- **Soft Teal**: `#14B8A6` (Teal-500) - Freshness, renewal

### Neutral Colors
- **Charcoal**: `#0F172A` (Slate-900) - Primary text
- **Slate**: `#475569` (Slate-600) - Secondary text
- **Light Gray**: `#F1F5F9` (Slate-100) - Backgrounds
- **White**: `#FFFFFF` - Pure backgrounds

### Gradient Combinations
1. **Primary Gradient**: `from-[#0B334A] via-[#0F4A6A] to-[#0B334A]`
2. **Accent Gradient**: `from-[#0B334A] to-[#DE3C3A]` (for urgent CTAs)
3. **Warm Gradient**: `from-amber-500 via-orange-500 to-amber-600`
4. **Cool Gradient**: `from-teal-500 via-cyan-500 to-teal-600`

## Typography

### Font Families
- **Primary**: Inter (Google Fonts) - Clean, modern, readable
- **Headings**: Inter with increased weight
- **Body**: Inter Regular (400)
- **Bold**: Inter SemiBold (600) and Bold (700)

### Font Sizes
- **Hero**: 4rem - 5rem (64px - 80px)
- **H1**: 3rem - 4rem (48px - 64px)
- **H2**: 2.25rem - 3rem (36px - 48px)
- **H3**: 1.875rem - 2.25rem (30px - 36px)
- **Body Large**: 1.25rem (20px)
- **Body**: 1rem (16px)
- **Small**: 0.875rem (14px)

## Spacing System
- **Base Unit**: 4px
- **Small**: 0.5rem (8px)
- **Medium**: 1rem (16px)
- **Large**: 1.5rem (24px)
- **XL**: 2rem (32px)
- **2XL**: 3rem (48px)
- **3XL**: 4rem (64px)

## Shadows
- **Small**: `0 1px 2px 0 rgba(0, 0, 0, 0.05)`
- **Medium**: `0 4px 6px -1px rgba(0, 0, 0, 0.1)`
- **Large**: `0 10px 15px -3px rgba(0, 0, 0, 0.1)`
- **XL**: `0 20px 25px -5px rgba(0, 0, 0, 0.1)`

## Border Radius
- **Small**: 0.5rem (8px)
- **Medium**: 1rem (16px)
- **Large**: 1.5rem (24px)
- **XL**: 2rem (32px)

## Icons
- **Library**: Lucide React
- **Size**: 20px - 24px for body, 32px - 48px for features
- **Color**: Inherit from text color or use accent colors

## Component Styles

### Buttons
- **Primary**: Indigo-600 background, white text, hover: Indigo-700
- **Secondary**: White background, indigo-600 text, border
- **Outline**: Transparent, indigo-600 border and text
- **Ghost**: Transparent, text color only

### Cards
- **Background**: White
- **Border**: Subtle gray border or shadow
- **Hover**: Elevate shadow, slight scale transform
- **Padding**: 1.5rem - 2rem

### Forms
- **Input Border**: Gray-300
- **Focus Border**: Indigo-500 with ring
- **Error**: Red-500
- **Success**: Emerald-500
